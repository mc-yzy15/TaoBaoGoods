# TaoBaoGoods C Version

## 状态

该目录现在是实验性手动辅助原型，不再作为自动购买主线维护。

- 不承诺自动登录、自动加购、自动提交订单。
- 主要作用是打开对应页面并在关键节点要求人工确认。
- 真正维护中的实现请使用仓库根目录说明里的 Python 主线。

## 目录

- `src/`: C 版实际源码。
- 根目录同名 `.c/.h` 文件：仅为兼容现有工程文件保留的转发层。
- `scripts/build.bat`: UTF-8 的本地构建脚本。
- 根目录旧 `build.bat`: 保留历史文件，不再作为权威入口。

## 使用方式

1. 编辑 `DefaultConfig.txt`。
2. 运行 `scripts/build.bat`。
3. 启动生成的可执行文件。
4. 按提示在浏览器中手动确认每个步骤。
